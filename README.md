# PowerPoint to PDF Converter

## Overview
The PowerPoint to PDF Converter tool allows users to seamlessly convert PowerPoint presentations (.pptx) into PDF format. This utility is particularly useful for ensuring that presentations maintain their formatting and are easily shareable.

## Features
- Batch conversion of multiple PowerPoint files.
- High-quality PDF output with preserved formatting.
- User-friendly command-line interface.

## Installation
To install the PowerPoint to PDF Converter, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/mayankkr001/powerpoint-to-pdf.git
cd powerpoint-to-pdf
pip install -r requirements.txt
```

## Usage
To convert a PowerPoint file to PDF, use the following command:

```bash
python convert.py <input_file.pptx> <output_file.pdf>
```

### Example
```bash
python convert.py presentation.pptx presentation.pdf
```
